# Personal_Dashboard
This project is Basic Personal Dashboard that brings together tasks like checking weather, news, find info about someone github, check the currency, and a joke to make it a little enjoyable. It is build using the apis of all the tasks that are included in it.
For the frontend python library Streamlit is used. It is a very easy and effective library for making frontend completely in python.
